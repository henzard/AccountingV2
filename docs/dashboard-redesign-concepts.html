Key UX Decisions

  • No cards — rows breathe with 12px padding
  • Hero R4,280 answers "can I spend?" instantly
  • 8px status dot replaces the busy fill-bar in the list
  • Fraunces serif makes the number feel weighty, important
  • IBM Plex Mono for all figures — tabular alignment
  • Single green accent (#00D68F) — no competing colors
  • Baby steps collapsed to 7-dot row, not a card
  • Score demoted to a side-row, not a badge

Key UX Decisions
